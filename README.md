# Hilight - 自定义文本划线高亮插件

Hilight是一款Edge浏览器插件，允许用户在网页上选择文本并进行自定义颜色高亮，让重要信息一目了然，提升阅读和学习效率。

## 功能特性

- ✨ **文本高亮**：在任何网页上选择文本后，自动显示高亮图标
- 🎨 **颜色选择**：提供多种高亮颜色供选择（黄色、橙色、红色、绿色、蓝色、紫色等）
- ⚙️ **全局开关**：通过简单的开关快速启用或禁用插件功能
- 📱 **响应式设计**：适配不同屏幕尺寸，在移动设备上也能良好工作

## 安装方法

1. 打开Edge浏览器
2. 访问 `edge://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本插件所在的文件夹
6. 插件安装完成，可以在浏览器右上角看到Hilight图标

## 使用说明

1. **高亮文本**：在网页中选择文本，会自动显示高亮图标，点击图标即可高亮所选文本
2. **选择颜色**：悬浮在高亮图标上，会弹出颜色选择面板，点击选择喜欢的颜色
3. **启用/禁用**：点击浏览器右上角的Hilight图标，在弹出的界面中可以快速开关插件
4. **设置默认颜色**：在设置页面中可以设置默认的高亮颜色

## 注意事项

- 本插件仅在Edge浏览器中测试过，其他浏览器可能需要适当修改
- 插件使用Chrome Extension API (Manifest V3)，需要支持Manifest V3的浏览器版本
- 高亮效果仅在当前页面会话中有效，刷新页面后高亮会消失（未来版本可能会添加保存功能）
